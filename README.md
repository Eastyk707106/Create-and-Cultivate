# Create & Cultivate

一个融合了机械动力自动化、应用能源2存储、农夫乐事烹饪与Alex生物冒险的Minecraft 1.20.1整合包。

## ✨ 特色
*   **核心模组**：Create (机械动力), Applied Energistics 2 (应用能源2), Farmer's Delight (农夫乐事) 及其附属。
*   **冒险扩展**：Alex's Mobs, Alex's Caves，保留原版生物特性与平衡。
*   **设计理念**：**无任何魔改**，最大化保留原版体验，专注于模组间的协同与探索乐趣。
*   **优化体验**：内置钠、铟等性能优化模组，确保流畅运行。

## 🚀 快速开始
1.  下载本页面最新的 `.zip` 或 `.mrpack` 发布文件。
2.  在您喜欢的启动器（如HMCL、PCL、CurseForge App）中，选择“导入整合包”。
3.  选择下载的文件，等待安装完成即可启动游戏。

## 📁 文件说明
- `mods/`：所有必需的模组文件。
- `config/`：预配置的模组设置，已针对平衡性和兼容性进行优化。
